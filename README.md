# cloud-web

This project is generated with [yo angular generator](https://github.com/yeoman/generator-angular)
version 0.15.1.

## Requirements

- Node.js
  - v6.10.0
- npm
  - 3.10.10
- Ruby
  - 2.2.0 or later
- Bundler
  - 1.7.10 or later

## Installation

```
bundle install
npm install -g grunt-cli bower
bower install
npm install
```
If yo command does not work...  
```
npm install -g yo
npm install -g generator-angular
```
